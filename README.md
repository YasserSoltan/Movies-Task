# Movies App

A simple and responsive movie management application built with Next.js and React. Add, edit, delete, and rate movies with an intuitive UI.

## Features

-  Browse movies with beautiful card layouts
-  Rate movies with star ratings
-  Edit movie details
-  Delete movies
-  Organize movies by genres
-  Fully responsive design
-  Perfect horizontal card alignment

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
